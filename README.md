## Sistema de Acompanhamento de Beneficiários BPC

O BPC Flow é um sistema desenvolvido em Python para organização e acompanhamento de beneficiários do Benefício de Prestação Continuada (BPC).

O projeto tem como objetivo facilitar o controle cadastral, acompanhamento da validade das informações e análise territorial dos beneficiários.

## Funcionalidades

✅ Cadastro de beneficiários

✅ Consulta e pesquisa por nome ou CPF

✅ Controle automático de atualização cadastral

✅ Identificação de:
- Cadastro atualizado
- Vencimento próximo
- Cadastro vencido

✅ Organização territorial:
- Itacuruçá
- Ilhas
- Marambaia
- Jaguanum
- Ilha de Itacuruçá

✅ Painel de acompanhamento

✅ Estatísticas

✅ Exportação para Excel

## Tecnologias utilizadas

- Python
- JSON
- OpenPyXL
